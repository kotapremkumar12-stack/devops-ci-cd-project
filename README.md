# DevOps CI/CD Project

## 🚀 Project Overview
This project demonstrates an end-to-end CI/CD pipeline using Jenkins, Docker, Git, and AWS EC2.

The goal of this project is to automate build, test, and deployment processes using DevOps best practices.

---

## 🛠 Tools & Technologies Used
- AWS EC2
- Jenkins
- Docker
- Git & GitHub
- Linux
- Maven
- Apache Tomcat

---

## ⚙️ Project Workflow
1. Developer pushes code to GitHub.
2. Jenkins automatically triggers build.
3. Maven builds the application.
4. Docker containerizes the application.
5. Application is deployed on AWS EC2.
6. Tomcat server runs the application.

---

## 📌 Outcome
Successfully implemented a CI/CD pipeline that reduces manual deployment effort and ensures continuous integration.
